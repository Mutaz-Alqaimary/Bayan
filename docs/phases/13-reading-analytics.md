# Phase 13 — Reading Analytics

## Goal
Turn `reading_sessions` history into trends a teacher (or student) can act on.

## Build
- Student progress charts
- WPM trends
- Accuracy trends
- Performance insights

## Definition of done
- Charts are correct in RTL (axis direction, legend placement, number formatting) — verify
  explicitly, don't assume the charting library handles it.
- Insights are written in plain, actionable language (e.g. "accuracy dropped over the last 3
  sessions"), not just raw numbers.
- Empty state for students/passages with no session history yet.
