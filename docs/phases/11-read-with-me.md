# Phase 11 — Read With Me

## Goal
The actual reading experience: a passage reader with an integrated vocabulary sidebar, designed
for sustained, comfortable Arabic reading.

## Build
- Passage reader
- Vocabulary sidebar
- Vocabulary lookup
- Word meanings
- Reading assistance experience

## Definition of done
- Reader typography follows `.claude/rules/design-system.md` (comfortable line-height,
  consistent Arabic typeface) — this view is the product's core value, treat it accordingly.
- Vocabulary lookup is fast and doesn't break reading flow (e.g. inline tap/click, not a full
  page navigation).
- Works correctly on mobile (touch-friendly), since students may read on phones/tablets.
- Feeds into the Phase 10 session workflow (starting/completing a reading session here).
